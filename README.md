# Словограй

Адаптивний застосунок із картками для вивчення англійської та іспанської мов.

## Можливості

- вхід через Google;
- особисті картки для кожного користувача;
- завантаження зображень;
- переклади та підказки з вимови;
- улюблені й вивчені картки;
- синхронізація через Supabase;
- світла та темна теми.

Сайт працює як статична сторінка та може бути опублікований через GitHub Pages.
